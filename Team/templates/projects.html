Projects

The club gives wide range of exposure when it comes to practical gain of knowledge. And therefore we believe in conducting regular club activity to help each member groom and nurture well. Some of which are mentioned below:

Hexapod

It was built using 3D printed parts.Each
of it's leg has three Servo motors, head and
tail also has a Servo motor.It is able to
Crawl in different terrains.

Dancing Robot

It was made for teaching purpose.
Different dance patterns were observed.

PID Controller

A robot that follows a line against a contrasting background by implementing Proportional Integral Derivative (PID) algorithm. It can traverse immensely complex networks that otherwise would be a nightmare if manually done.

Robotic Arm

It is made using self designed 3D printed parts and has 3 degrees of freedom.

Maze Solver

Using PID Controlled motion and Path planners it scans the whole arena and then it finds the shortest path using a particular algorithm.

Badminton Bot

Badminton is a dynamic sport and
therefore only advanced concepts and technologies are in order to realize a badminton robot. Based on many projects with industrial partners.

Pick and Place

It was built using 3D printed parts and controlled by Servo motors. It was able to pick a tennis ball.

Snake Robot

We have Fabricated a snake type robot
with 12 Servo motors and implemented sine and cosine waves to see Serpentile and Rectilinear motion.

Wall Follower

It is a simple project made with the help of sensors.

Quadruped Robot

It has been fabricated using 3D printed Parts.It consists of 12 Servo Motors.

Survelliance Robot

Built a robot car with an On-board pan-tilt camera, which sends live video feeds to the webpage and the bot can be controlled using the web interface.

Self Balancing Robot

It was made using 3D printed parts,
Stepper motors, ATmega IC and MPU6050. The main concept behind the self balancing bot was in the code.
